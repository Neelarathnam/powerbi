# powerbi
This project analyzes retail sales data using Power BI to identify revenue trends, profit gaps, and regional performance. Data is cleaned and modeled using a star schema, DAX measures are applied for key metrics, and an interactive dashboard supports data-driven decisions.
Retail Sales Performance Analysis

# Project Overview
This project analyzes retail sales data using Power BI to evaluate sales performance, profit margins, and regional trends. The goal is to identify profitable and non-profitable products and support data-driven decision-making for promotions and inventory planning.

# Files Included in This Repository

# Retail_Sales_Performance_Analysis.pdf
Contains the complete project report, including overview, step-by-step process, data cleaning, data modeling, DAX measures, dashboard screenshots, insights, and business recommendations.

# Retail_Sales_Analysis.pbix
Power BI file containing the data model, DAX calculations, and the interactive overview dashboard.

# Retail_Sales_Dataset.xlsx / .csv
Raw dataset used for analysis and dashboard creation.

# Tools and Technologies Used

Power BI
Power Query
DAX (Data Analysis Expressions)
Microsoft Excel (for dataset)

# Key Features
Data cleaning and validation
Star schema data modeling
DAX measures for Sales, Profit, and Profit Margin
Interactive overview dashboard with filters
Regional and category-wise performance analysis

# How to Use
Download or clone the repository.
Open the .pbix file using Power BI Desktop.
Review the dashboard and interact with slicers.
Refer to the PDF for detailed step-by-step explanation and insights.

# Conclusion
This project demonstrates the use of Power BI for transforming raw retail data into meaningful insights through effective data modeling, calculations, and visualization.
